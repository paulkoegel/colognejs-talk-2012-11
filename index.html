Global State

Set data-state="green" on a slide and "green" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background and headline styles.

  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) ) {

    };
